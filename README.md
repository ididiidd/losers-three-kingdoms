# 失敗者的三國（觀眾版）

搭配 YouTube 頻道《失敗者的三國》的互動式時間軸瀏覽網頁，配合節目進度同步更新。

## 觀眾可以做什麼

- 👀 瀏覽當前節目進度的人物名牌與失敗時間軸
- 🔍 搜尋人物名稱
- 🎚 用篩選功能：指定顯示章回範圍、隱藏已退場或中離的人物
- 🖱 拖曳調整名牌順序（純個人樂趣，重新整理後恢復）
- 📖 點兩下任何名牌查看人物資訊卡（登場回、失敗紀錄清單）

## 觀眾不能做什麼

無法新增、編輯、或刪除任何資料。所有內容由頻道主更新。

## 資料更新流程（頻道主用）

1. 在編輯版網站新增 / 修改資料
2. 打開篩選面板 → 匯出 JSON
3. 把下載的檔案重新命名為 `data.json`
4. 上傳到這個 repo 覆蓋原本的 `data.json`
5. Vercel 自動部署（約 30 秒）
6. 觀眾重新整理頁面即可看到新進度

## 技術說明

- 純靜態 HTML，無後端
- 頁面載入時 fetch `./data.json` 讀取資料
- 使用 `?t=timestamp` query string 避免瀏覽器快取
